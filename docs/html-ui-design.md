# HTML UIデザインノート

## 目的
- HTML/CSS/JSの変更を段階的に進めるための設計メモ。
- まず現行デザインをベース(v0)として固定し、その上に改善を積み上げる。

## ベース(v0)保存（現行）
- 画面は `header / guide / controls / status / results` の並び。
- 背景は薄いグレー、カードは白。
- 上部にタブ（動画/テーマ）と検索・並び替えがある。
- 結果カードにはタイトル、日付、タグ、大見出し、小見出しを表示。

## 今回のv1方針
- 地の説明文は削除。
- 全体背景を薄緑にする。
- UIを2ブロックに整理する。
  1. 上部: トーク集カード群（タップ展開・スクロールで自動折り畳み）
  2. 下部: 操作メニュー（通信バー、展開/折り畳み、検索）
- 上部/下部とも共通の白カード背景を使う。

## 上部トーク集カード群（仕様）
- 折り畳み時のヘッダ構成:
  - 1列目: タイトル（動画URLリンク）
  - 2列目: 投稿日付 / 大見出し件数（`◯件`）
  - 右側: 小さなサムネイル
- タップで展開。
- 展開時の表示順:
  1. 大見出し（`▶` で小見出しを開閉）
  2. 自動抽出タグ
- 大見出し文言はリンク（タイムスタンプURL）を使う。
- 小見出し文言はリンクを付けない。
- 1ページ分スクロールで自動折り畳み。

## 下部操作メニュー（仕様）
- 上段: 通信状態バー + 展開ボタン + 折り畳みボタン
- 下段: 検索テキストボックス
- 検索仕様:
  - 通常検索: タイトル / 大見出し / 小見出し
  - `#xxx` 検索: タグのみ対象

## 変更管理ルール
- 見た目変更はこのファイルに先に追記してから実装する。
- 大きな変更は避け、小さな単位で進める。
